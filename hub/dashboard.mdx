The dashboard shows your server's health and resources in real-time.

## Main Dashboard

When you open Hub, you'll see:

- **CPU** - Current usage percentage
- **Memory** - How much RAM is used
- **Disk** - Storage space used
- **Network** - Traffic in/out (bytes per second)
- **Load Average** - Server load (1, 5, 15 min)

All metrics update every **10 seconds**.

## Historical Charts

Click any metric card to see historical data. Choose time range:

- 1 hour
- 6 hours
- 24 hours
- 7 days (Pro plan)

## Containers

View and manage your Docker containers:

- See all containers (running and stopped)
- Check CPU and memory usage per container
- Stream logs in real-time
- Start, stop, or restart containers with one click

<Info>
  Docker containers appear automatically if Docker is installed on your server.
</Info>

## Services

Hub automatically detects common services running on your server:

- **Web**: nginx, Apache
- **Database**: PostgreSQL, MySQL, MongoDB, Redis
- **Other**: Docker, SSH

Each service shows whether it's running or stopped.

## Network

**Connections** - See active network connections with remote addresses and which process created them.

**Listening** - See which ports your server is listening on and what process is using each port.

## Settings

View your current plan, license status, and data retention info.

## Theme

Switch between **dark** and **light** mode using the theme button in the sidebar.

## Data Retention

Old metrics are automatically cleaned up based on your plan:

| Plan | Data kept for |
|------|---------------|
| Personal | 24 hours |
| Pro | 30 days |
