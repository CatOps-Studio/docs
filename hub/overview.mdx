CatOps Hub is a lightweight server monitoring dashboard that you install on your server. Monitor CPU, memory, disk, network, and Docker containers from a beautiful web interface.

## What You Get

<CardGroup cols={2}>
  <Card title="Real-time Metrics" icon="chart-line">
    CPU, memory, disk usage and network traffic updated every 10 seconds
  </Card>
  <Card title="Docker Management" icon="docker">
    View containers, check logs, start/stop/restart with one click
  </Card>
  <Card title="Service Detection" icon="server">
    Automatically detects nginx, PostgreSQL, Redis, and other services
  </Card>
  <Card title="Network Connections" icon="network-wired">
    See active connections and listening ports on your server
  </Card>
</CardGroup>

## Quick Start

<Steps>
  <Step title="Install">
    Run the install script on your server:
    ```bash
    curl -fsSL https://get.catops.app/hub.sh | sh
    ```
  </Step>
  <Step title="Activate License">
    Start Hub and enter your license key:
    ```bash
    catops-hub serve
    ```
  </Step>
  <Step title="Open Dashboard">
    Access the dashboard in your browser:
    ```
    http://your-server-ip:8090
    ```
  </Step>
</Steps>

## Requirements

- **Linux**, macOS, or FreeBSD
- Root access for installation
- Port **8090** (or custom port)
