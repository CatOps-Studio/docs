## What is CatOps?

CatOps is a suite of tools for server monitoring and management:

- **Hub** - Self-hosted server monitoring dashboard
- **CLI** - Command-line tools for server management

## Hub Installation

```bash
curl -fsSL https://get.catops.app/hub.sh | sh
```

After installation:
1. Open `http://your-server-ip:8090`
2. Create your admin account
3. Activate your license

## CLI Installation

```bash
curl -fsSL https://get.catops.app/cli.sh | sh
```
