CatOps CLI is a command-line tool for server management and productivity.

## Features

<CardGroup cols={2}>
  <Card title="SSH Management" icon="key">
    Generate, list, and manage SSH keys
  </Card>
  <Card title="Quick Connect" icon="plug">
    SSH shortcuts with server aliases
  </Card>
  <Card title="System Info" icon="desktop">
    Quick system information commands
  </Card>
  <Card title="Cross-platform" icon="laptop">
    Works on Linux, macOS, and Windows
  </Card>
</CardGroup>

## Quick Install

```bash
curl -fsSL https://get.catops.app/cli.sh | sh
```

## Basic Usage

```bash
# Show help
catops --help

# Show version
catops --version
```
