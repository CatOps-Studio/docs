CatOps CLI is a lightweight monitoring agent that runs on your servers and sends metrics to the cloud dashboard.

## Features

<CardGroup cols={2}>
  <Card title="System Monitoring" icon="chart-line">
    CPU, memory, disk, and network metrics
  </Card>
  <Card title="Telegram Alerts" icon="bell">
    Get notified when something goes wrong
  </Card>
  <Card title="AI Assistant" icon="robot">
    Ask questions about your server
  </Card>
  <Card title="Auto-start" icon="play">
    Runs automatically on boot
  </Card>
</CardGroup>

## Quick Install

```bash
curl -fsSL https://get.catops.app/install.sh | bash
```

With cloud mode (for Telegram alerts):

```bash
curl -fsSL https://get.catops.app/install.sh | AUTH_TOKEN="your_token" bash
```

<Warning>
  Do NOT use `sudo` with the install script. It installs to your home directory.
</Warning>

## Basic Usage

```bash
# Check status
catops status

# Ask AI about your server
catops ask "why is CPU high?"

# Configure thresholds
catops set cpu 80
catops set mem 80
catops set disk 90

# Restart monitoring
catops restart
```

## Difference from Hub

| | Hub | CLI |
|-|-----|-----|
| **Purpose** | Web dashboard | Monitoring agent |
| **Runs on** | Your server | Your server |
| **Interface** | Web browser | Command line |
| **License** | Required ($5-12/mo) | Free |
| **Features** | Dashboard, Docker, Network | Metrics, Alerts, AI |
